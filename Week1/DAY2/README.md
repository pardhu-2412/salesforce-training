# 🚀 Day 2 - Salesforce Platform Basics

Welcome to Day 2 of my Salesforce learning journey.

---

# ☁️ 1. What is Salesforce Platform?

Salesforce Platform is a cloud-based platform used to build business applications. It helps companies manage data, automate tasks, and improve customer services. Salesforce Platform allows users to create apps without needing advanced coding knowledge.

---

# 🧩 2. Important Salesforce Terms

## 📱 App
An App in Salesforce is a collection of tools, tabs, and features used for a specific purpose. It helps users access related information in one place.

### Example:
Sales App, Service App, Student Management App

---

## 📦 Object
An Object is used to store data in Salesforce. It is similar to a table in a database.

### Example:
- Student Object
- Employee Object
- Product Object

Objects store records and information.

---

## 🗂️ Tab
A Tab is used to open and access objects, reports, or dashboards in Salesforce. It helps users navigate easily inside the application.

### Example:
Accounts Tab, Contacts Tab, Reports Tab

---

# ⚙️ 3. Configuration vs Coding

| Configuration | Coding |
|---|---|
| Uses drag-and-drop tools | Uses programming languages |
| Easy to create features | Requires coding knowledge |
| Faster development | More customization |
| No-code or low-code | Full-code development |

### Simple Explanation
Configuration is used for simple customizations without writing code. Coding is used when advanced features and complex logic are needed.
