# 🚀 Day 4 – Flow Builder

# 📘 1. What is Flow Builder?

Flow Builder is a Salesforce automation tool used to automate business processes without writing code. It helps users create workflows, collect data, send emails, update records, and guide users through screens using a graphical interface.

Flow Builder improves productivity by reducing manual work and making business operations faster and more accurate. It is widely used in enterprise systems for process automation.

---

# 🔄 2. Types of Flows

# 🖥️ Screen Flow

Screen Flow is an interactive flow that displays screens to users and collects input from them. It is mainly used for forms, surveys, data entry, and guided processes.

## ✨ Features of Screen Flow
- Interacts with users through screens
- Collects and displays data
- Can create or update records
- Used for step-by-step processes

## 📌 Example
A student registration form where users enter personal and course details.

---

# ⚙️ Record-Triggered Flow

Record-Triggered Flow runs automatically when a record is created, updated, or deleted in Salesforce. It is used for automating backend business operations.

## ✨ Features of Record-Triggered Flow
- Runs automatically in the background
- No user interaction required
- Updates records and sends notifications
- Improves process efficiency

## 📌 Example
Automatically assigning a lead to a sales representative when a new lead is created.

---

# 💡 3. Automation Ideas

## 📍 1. Automatic Lead Assignment
When a new lead is created, Salesforce automatically assigns it to the correct sales representative based on region or department.

---

## 📍 2. Student Attendance Notification
If a student’s attendance falls below a certain percentage, an automated email notification is sent to the student and faculty.

---

## 📍 3. Order Confirmation Process
When a customer places an order, the system automatically sends an order confirmation email and updates inventory records.

---

## 📍 4. Employee Leave Approval
Employees submit leave requests using a screen flow, and managers receive approval notifications automatically.

---

## 📍 5. Support Ticket Escalation
If a support ticket is not resolved within a specified time, the system automatically escalates it to a higher-level support team.

---

# ⚖️ 4. Manual vs Automated Process

| Manual Process | Automated Process |
|---|---|
| Requires human effort | Runs automatically |
| Time consuming | Saves time |
| Higher chance of errors | Reduces errors |
| Repetitive work | Efficient workflow |
| Slower communication | Faster notifications |
| Difficult to track | Easy monitoring and tracking |

---

# 🌟 5. Reflection – Why Automation Matters in Enterprise Systems

Automation is important in enterprise systems because it improves efficiency, accuracy, and productivity. Businesses handle large amounts of data and repetitive tasks every day. Manual processing can lead to delays and mistakes.

Using Flow Builder, organizations can automate approvals, notifications, record updates, and customer interactions. Automation reduces workload for employees and allows them to focus on important business activities.

Enterprise automation also improves customer experience by providing faster responses and consistent processes. Overall, automation helps organizations save time, reduce costs, and increase operational performance.
